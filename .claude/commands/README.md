# `.claude/commands`

Project command shortcuts currently defined:

- `/pipeline-build [optional-focus]`
- `/preview-build [job-id]`
- `/trace-footer [job-id]`
- `/trace-shared-chrome [job-id]`
- `/trace-spectra [optional-block-name]`

These commands are scoped to the `vibepress` workflow and target recurring `ai-pipeline` debugging paths:

- validating NestJS pipeline changes
- building generated preview frontends
- tracing footer/shared chrome data contracts
- auditing Spectra/UAGB migration support gaps
